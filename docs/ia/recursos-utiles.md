## Claude Code

### `/model opusplan` — Opus planifica, Sonnet ejecuta

> **@dani_avila7 en X** · [Ver post](https://x.com/dani_avila7/status/2033573729687478649)

Alias híbrido de Claude Code que combina lo mejor de ambos modelos:

- **Plan mode** → usa **Opus** para razonamiento complejo y diseño de soluciones
- **Ejecución** → usa **Sonnet** para escribir el código de forma rápida y eficiente

**Cómo usarlo:**

```bash
/model opusplan
```

Ideal para tareas que requieren pensar bien la arquitectura antes de codear. Opus piensa, Sonnet construye.
