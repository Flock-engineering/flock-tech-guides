# NetCore — Estándares y Linteo

Para NetCore usamos **Visual Studio** como IDE principal.

---

## Formateo

Usamos el **Code Cleanup on Save** integrado de Visual Studio, que aplica las especificaciones de formato automáticamente al guardar.

**Activación:**

## Linteo

### SonarLint (Visual Studio)

Usamos **SonarLint** como herramienta de linteo, igual que en Java. Analiza el código en tiempo real y sugiere correcciones.

**Instalación:**

Instalar desde el [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=SonarSource.SonarLintforVisualStudio2022).
